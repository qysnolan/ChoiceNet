Form

  • Change Form Title/Description: Part 1

    Click the "Options/Help" menu button to open the dropdown menu.

    Then select "Edit Title/Description" from the dropdown options.

  • Change Form Title/Description: Part 2

    Enter in the new Form title and description then hit "Save".

Standards

  • Add a Standard

    First start by adding a standard.

  • Delete a Standard

    Open the Standards info panel by hitting the info icon as in Step 1 of Edit a Standard. The delete standard button is located in the Standards info panel.

    Warning: this will delete all indicators, elements, descriptors underneath the standard.

  • Edit a Standard: Step 1

    After adding a standard you'll be at the following screen.

    1) To edit the standard's details hit the info button to the right of the "Add Standard" button.

    2) Click the "Edit" button to pull up the edit standard screen.

  • Edit a Standard: Step 2

    Make your changes to the standard and then hit "Save" when you're done.

Indicators

  • Add an Indicator

    Click the large "Add Indicator" button.

  • Delete an Indicator

    Just click the "Delete" button for the indicator you would like to remove.

    Warning: this will delete all elements and descriptors underneath this indicator.

  • Edit an Indicator: Part 1

    Click the "Edit" button found on the indicator you'd like to edit.

  • Edit an Indicator: Part 2

    Make your changes to the indicator and then hit "Save".

Elements

  • Add an Element

    1) Make sure you've opened up the accordion element for the indicator by clicking on it.

    2) Then click "Add Element".

  • Delete an Element

    Just click "Delete" on the desired element.

    Warning: this will delete all the descriptors attached to this element.

  • Edit an Element: Part 1

    Click the "Edit" button for the element you'd like to edit.

  • Edit an Element: Part 2

    Make the desired changes to the Element field then hit "Save".

  • Adding Descriptors: Part 1

    Adding descriptors is done as apart of editing an element.

    Click "Add Descriptor" to begin.

  • Adding Descriptors: Part 2

    Now that you've added a desriptor, you can see fields have been added so that you can modify it's information.

    Simply make the changes you'd like and then hit "Save".

  • Delete a Descriptor

    If you'd like to delete a descriptor, simply click the delete button next to it's name in the descriptor fields section.
